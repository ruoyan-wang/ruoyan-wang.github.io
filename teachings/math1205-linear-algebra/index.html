Linear Algebra

This course introduces the core ideas of linear algebra, including systems of linear equations, vector spaces, matrices, determinants, eigenvalues, eigenvectors, and linear transformations.

Instructor: Prof. Zhihao Ma

Term: Fall

Location: Main Campus, Room 1205

Time: Mondays and Wednesdays, 10:00-11:30 AM

Course Overview

This course covers the foundations of linear algebra. Students will learn how to:

  • Solve systems of linear equations using matrix methods
  • Work with vector spaces, bases, and dimensions
  • Understand linear transformations and their matrix representations
  • Compute determinants, eigenvalues, and eigenvectors
  • Apply linear algebra tools to problems in engineering, computing, and data analysis

Prerequisites

  • Basic calculus
  • Comfort with algebraic manipulation
  • Prior exposure to vectors is helpful but not required

Textbooks

  • Linear Algebra and Its Applications by David C. Lay
  • Introduction to Linear Algebra by Gilbert Strang

Grading

  • Homework: 30%
  • Midterm: 30%
  • Final Exam: 35%
  • Participation: 5%
